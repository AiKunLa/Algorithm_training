## 主要思想
for循环横向遍历，递归纵向遍历，回溯不断调整结果集

递归函数一般没有返回值 
void back(){
    if(终止条件){
        存放结果;
        return;
    }
    for(遍历集合){
        // 剪枝
        if(剪枝条件)break;
        处理节点; 
        back();// 递归
        回溯; // 回溯的过程，撤销处理结果
    }
    return;
}
